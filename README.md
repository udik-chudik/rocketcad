# RocketCAD

This program allows to simulate a rocket flight, perform construction and automatic control optimization.

> Note: the program is under heavy development

## How to run it?

1. Install the [Miniconda](https://conda.io/miniconda.html)
2. Install the required packages:
```
conda install numpy
conda install scipy
conda install matplotlib`
```
Download and install [transforms3d](https://pypi.python.org/pypi/transforms3d) package

3. Just run it:
```
cd rocketcad
python main.py
```
